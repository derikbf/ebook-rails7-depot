# Project Depot
The project was built following the business rules from the Book.

## Technology Stack

- ruby on rails ``7.0.4``
- ruby ``3.1.0``
- sqlit3 ``1.4``
- TailwindCSS

## Set up

1. Clone the project
```
git clone https://github.com/derikbf/ebook-rails7-depot
```
```
cd depot
```
2. Install dependencies
``` 
bundle
``` 
3. Start the project
```
bin/rails server
```
4. http://localhost:3000/
